Name

ambient()
Examples 
size(100, 100, P3D)
background(0)
noStroke()
directionalLight(153, 153, 153, .5, 0, -1)

ambient(51, 26, 0)

sphere(30)
Description 画面に描画された図形の環境反射率を設定します。 これは環境光成分と組み合わされます。 パラメータで設定された色成分は反射率を定義します。 たとえば、デフォルトのカラーモードにおいて、v1 = 255、v2 = 127、v3 = 0に設定すると、すべての赤色光が反射し、緑色光の半分が反射します。 ある形状のマテリアルプロパティを設定するときに、 emissive（）、 specular（）、および shininess（）と組み合わせて使用します。
Syntax
ambient(rgb)
ambient(gray)
ambient(v1, v2, v3)
Parameters
rgbint: any value of the color datatype
grayfloat: number specifying value between white and black
v1float: red or hue value (depending on current color mode)
v2float: green or saturation value (depending on current color mode
v2float: blue or brightness value (depending on current color mode)
