Name

+= (add assign)
Examples 
a = 50
b = 23
a += b  # Sets 'a' to 73
c = "c"
c += "d"  # Sets 'c' to "cd"
d = [1, 2, 3]
d += [4]  # Sets 'd' to [1,2,3,4]
Description 加算と数値の割り当て、文字の連結と文字列の割り当て、およびリストの追加とリストの割り当てを組み合わせたものです。 a += bは、a = a + bと等しいです。
Syntax
value1 += value2
Parameters
value1int, float, list, string
value2any value the same datatype as value1
Related = (assign)
+ (addition)
-= (subtract assign)
