Name

+ (addition)
Examples 
a = 50 + 5  # Sets 'a' to 55
b = a + 5  # Sets 'b' to 60

s1 = "Chernenko"
s2 = "Brezhnev"
sc1 = s1 + s2
sc2 = s1 + ", Andropov, " + s2
print(sc1)  # Prints "ChernenkoBrezhnev"
print(sc2)  # Prints "Chernenko, Andropov, Brezhnev"

s1 = "Gorbachev"
i = 1987
sc1 = s1 + str(i)
print(sc1)  # Prints "Gorbachev1987"
Description 2つの値を加算するまたは文字列を連結します。数学演算子として、2つの値の合計を計算します。 文字列演算子として、2つの文字列を1つに結合します。 連結では、文字列以外のデータ型はstr()を使用して文字列にキャストする必要があります。
Syntax
value1 + value2
Parameters
value1String, int, float, char, byte, boolean
value2String, int, float, char, byte, boolean
