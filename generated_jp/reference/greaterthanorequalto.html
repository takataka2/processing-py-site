Name

>= (greater than or equal to)
Examples 
a = 23
b = 23
if a >= b:
    print("variable a is greater or equal to variable b ")
Description Tests if the value on the left is larger than the value on the right or if the values are equivalent.
Syntax
value1 >= value2
Parameters
value1int or float
value2int or float
