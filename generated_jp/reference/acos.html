Name

acos()
Examples 
a = PI
c = cos(a)
ac = acos(c)
# Prints "3.1415927 : -1.0 : 3.1415927"
print("%.7f : %.1f : %.7f" % (a, c, ac))

a = PI + PI/4.0
c = cos(a)
ac = acos(c)
# Prints "3.926991 : -0.70710665 : 2.3561943"
print("%.6f : %.8f : %.7f" % (a, c, ac))
Description cos（）の逆数、値のアークコサイン（逆コサイン）を返します。 この関数は、-1から1の範囲の引数を想定し、返り値は 0 から PI（3.1415927）の範囲で返されます。
Syntax
acos(value)
Parameters
valuefloat: the value whose arc cosine is to be returned
