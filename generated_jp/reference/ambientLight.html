Name

ambientLight()
Examples 
size(100, 100, P3D)

# the ambient light changes their color.

Description 環境光を追加します。 環境光は特定の方向から来るのではなく、光線は周囲を非常に跳ね返っているので、オブジェクトは四方から均等に照らされています。 環境光は、ほとんどの場合、他の種類の光と組み合わせて使用されます。 ループするプログラムの中でで光の状態を維持するには、光を設定するプログラムを draw()に含める必要があります。 それらをループしているプログラムの setup()に配置すると、ループを最初に実行したときにのみ効果があります。 v1 、 v2 、および v3 パラメータは、現在のカラーモードに応じて、RGB値またはHSB値として解釈されます。
Syntax
ambientLight(v1, v2, v3)
ambientLight(v1, v2, v3, x, y, z)
Parameters
v1float: red or hue value (depending on current color mode)
v2float: green or saturation value (depending on current color mode)
v3float: blue or brightness value (depending on current color mode)
xfloat: x-coordinate of the light
yfloat: y-coordinate of the light
xfloat: z-coordinate of the light
