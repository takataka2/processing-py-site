Name

arc()
Examples

Description スクリーンに弧を描きます。 弧は a 、 b 、 c 、および d パラメータで定義された楕円の外縁に沿って描画されます。 。 弧の楕円の原点は ellipseMode()関数で変更できます。 弧を描く角度（ラジアン）を指定するには、 start および stop パラメータを使用します。
弧を描くには3つの方法があります。 使用されるレンダリング手法は、オプションの7番目のパラメータによって定義されます。 上記の例に示されている3つのオプションは、PIE、OPEN、およびCHORDです。 デフォルトのモードは、PIE塗りのOPENストロークです。
Syntax
arc(a, b, c, d, start, stop)
arc(a, b, c, d, start, stop, mode)
Parameters
afloat: x-coordinate of the arc's ellipse
bfloat: y-coordinate of the arc's ellipse
cfloat: width of the arc's ellipse by default
dfloat: height of the arc's ellipse by default
startfloat: angle to start the arc, specified in radians
stopfloat: angle to stop the arc, specified in radians
